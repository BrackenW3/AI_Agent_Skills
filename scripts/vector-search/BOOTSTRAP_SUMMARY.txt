================================================================================
BOOTSTRAP STATUS CHECKS
================================================================================

The bootstrap.py script validates:

1. pip / Package Manager
   - Checks pip --version
   - Verifies pip can install packages

2. Dependencies Installation
   - Installs all 8 packages from requirements.txt
   - Verifies no errors during installation

3. Package Imports
   - Tests import of: openai, supabase, tiktoken, docx, pypdf, aiohttp, dotenv, pydantic
   - Identifies any missing packages

4. Environment File
   - Checks .env exists in current directory
   - Loads variables from .env

5. Required Variables
   - Verifies: AZURE_OPENAI_API_KEY, AZURE_OPENAI_ENDPOINT, EMBEDDING_MODEL,
     SUPABASE_URL, SUPABASE_API_KEY
   - Reports any missing values

6. Variable Format Validation
   - AZURE_OPENAI_ENDPOINT must be HTTPS URL
   - SUPABASE_URL must contain ".supabase.co"
   - API keys must be non-empty and reasonable length

7. Azure OpenAI Connectivity
   - Creates client with API key and endpoint
   - Tests embedding generation on sample text
   - Returns embedding dimension (1536 for text-embedding-3-small)

8. Supabase Connectivity
   - Creates Supabase client
   - Queries documents table (tests connection)
   - Verifies API credentials

9. Supabase Schema Check
   - Verifies documents table exists
   - Verifies file_metadata table exists
   - Confirms pgvector extension is enabled

10. Directory Scanning
    - Scans configured paths (OneDrive, Google Drive, LOCAL_PATHS)
    - Counts total files
    - Breaks down by source

11. Cost Estimation
    - Calculates tokens needed for all files
    - Applies Azure pricing ($0.02 per 1M tokens)
    - Estimates total cost
    - Estimates indexing time (default 0.5s per file)
