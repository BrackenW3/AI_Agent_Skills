# Mistral AI Skills

Skills specific to **Mistral AI** models (Mistral 7B, Mixtral, Mistral Large, etc.).

## Configuration

Requires the following environment variable (see `config/vscode/.env.template`):

```
MISTRAL_API_KEY=<your-mistral-api-key>
```
