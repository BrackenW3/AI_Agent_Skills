# Tools Overview

This directory contains shared tools and integrations used by AI agent skills.

## Structure

```
tools/
└── mcp/    # Model Context Protocol (MCP) server configurations
```

## Model Context Protocol (MCP)

MCP is an open protocol that enables AI models to interact with external tools, data sources, and services in a standardized way. See [tools/mcp/README.md](mcp/README.md) for details.
